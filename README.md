# 셀러보드 (Sellerboard) 🛍️

> 쇼핑몰 상품 정보를 효율적으로 수집하고 관리하는 크롬 확장프로그램

## ✨ 주요 기능

### 기본 기능 (MVP)
1. **📦 상품 수집**: 상품 상세 페이지에서 상품명, 가격, 이미지, 옵션, 재고 등 자동 수집
2. **🏪 상점 몰털이**: 상점 전체 상품을 자동으로 일괄 수집
3. **🎯 영역 드래그 선택**: 화면에서 원하는 영역을 드래그하여 상품 선택 및 수집

### 확장 기능
4. **🏷️ 자동 분류 및 태깅**: 수집된 데이터를 자동으로 분류하고 태그 부여
5. **📈 실시간 가격 변동 알림**: 가격/재고 변동 시 알림
6. **📤 다양한 내보내기**: CSV, JSON, Google Sheets, Notion 등
7. **⏰ 스케줄러**: 예약 크롤링 기능
8. **👤 사용자 프로필**: 사이트별 맞춤 수집 프로필 저장
9. **📊 관리 대시보드**: 수집 데이터 검색, 필터, 정렬, 통계
10. **🔧 예외 처리 강화**: 로딩 지연, 동적 콘텐츠 대응
11. **⚙️ 커스터마이징**: 커스텀 스크립트 및 위젯 설정

## 🚀 설치 방법

### 1. 확장프로그램 로드
1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택

### 2. 아이콘 추가 (선택사항)
아이콘 이미지를 `assets/icons/` 폴더에 추가:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

## 📖 사용 방법

### 상품 수집
1. 상품 상세 페이지 방문
2. 페이지 우측 하단의 셀러보드 위젯 클릭
3. "📦 상품 수집" 버튼 클릭
4. **단축키**: `Alt + S`

### 영역 드래그 선택
1. 위젯에서 "🎯 영역 드래그 선택" 버튼 클릭
2. 마우스로 원하는 영역 드래그
3. 선택된 상품들이 자동 수집
4. **단축키**: `Alt + D` (모드 전환), `ESC` (모드 해제)

### 상점 몰털이
1. 상점 페이지 방문
2. 위젯에서 "상점 몰털이" 토글 활성화
3. 자동으로 모든 상품 수집 시작
4. 진행 상황이 위젯에 표시됨

### 대시보드
1. 확장프로그램 아이콘 클릭
2. "📊 대시보드 열기" 버튼 클릭
3. 수집된 상품 관리, 검색, 필터, 정렬
4. CSV/JSON으로 내보내기

## 🏗️ 프로젝트 구조

```
sellerboard/
├── manifest.json              # 확장프로그램 설정
├── background/
│   └── service-worker.js      # 백그라운드 작업, 스케줄러
├── content/
│   ├── content-script.js      # 메시지 중계, 초기화
│   ├── widget.js              # 플로팅 위젯 UI
│   └── drag-selector.js       # 드래그 선택 기능
├── popup/
│   ├── popup.html             # 팝업 UI
│   ├── popup.js               # 팝업 로직
│   └── popup.css              # 팝업 스타일
├── dashboard/
│   ├── dashboard.html         # 대시보드 UI
│   ├── dashboard.js           # 대시보드 로직
│   └── dashboard.css          # 대시보드 스타일
├── lib/
│   ├── storage-manager.js     # IndexedDB 관리
│   └── parser.js              # 상품 데이터 추출
├── styles/
│   ├── widget.css             # 위젯 스타일
│   └── overlay.css            # 드래그 오버레이 스타일
└── assets/
    └── icons/                 # 아이콘 이미지
```

## 🔧 기술 스택

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Storage**: IndexedDB + Chrome Storage API
- **Background**: Service Worker (Manifest V3)
- **UI Design**: 글래스모피즘, 그라데이션, 다크모드 지원

## ⚙️ 설정

### Rate Limiting (요청 간격)
기본값: 2000ms (2초)
- Service Worker에서 `rateLimitDelay` 설정으로 조정 가능
- 서버 부하 방지를 위해 최소 1초 권장

### 저장소 용량
- Chrome Storage: 최대 10MB (기본 설정용)
- IndexedDB: 무제한 (상품 데이터용)

## 🔐 법적 고려사항

> ⚠️ **중요**: 웹 크롤링 사용 시 반드시 준수해야 할 사항입니다.

## 🐛 문제 해결

### 상품 수집이 안될 때
1. 페이지를 새로고침 후 다시 시도
2. 지원하지 않는 플랫폼일 수 있음 (Issue 등록 부탁)
3. 개발자 도구(F12) 콘솔에서 오류 확인

### 위젯이 보이지 않을 때
1. 확장프로그램이 활성화되어 있는지 확인
2. 페이지를 새로고침
3. `chrome://extensions/`에서 "오류" 확인
